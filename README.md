# DotFiles

- zsh
- bash
- powershell

# Install

```bash
$ git clone https://github.com/airtonix/dotfiles ~/.dotfiles
```

### bash

```bash
$ ~/.dotfiles/bash/setup
```

### zsh
1. install zsh
2. set it as your shell `chsh $(which zsh)`
3. log out
4. log in
5. link the configuration files

```bash
$ ~/.dotfiles/zsh/setup
```
