# DotFiles

- zsh
- powershell

# Install

```bash
$ git clone https://github.com/airtonix/dotfiles ~/.dotfiles
```

### zsh
1. install zsh
2. set it as your shell `chsh $(which zsh)`
3. log out
4. log in
5. link the configuration files

```bash
$ ~/.dotfiles/zsh/setup
```

### powershell

```bash
PS> . ~/.dotfiles/ps/provision.ps1
PS> . ~/.dotfiles/ps/setup.ps1
```
