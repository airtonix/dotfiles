# Install

## From Web

you can just let the web install do its thing...

**linux**

```bash
curl -fsSL https://get.comtrya.dev | sh
comtrya https://github.com/airtonix/dotfiles
```

**macos**

```bash
curl -fsSL https://get.comtrya.dev | sh
comtrya https://github.com/airtonix/dotfiles
```

**windows**

```powershell
iwr "https://get.comtrya.dev/ps" -UseBasicParsing | iex
comtrya https://github.com/airtonix/dotfiles
```
